# **App Name**: Carpool2Campus

## Core Features:

- User Profile: User profile creation and management (name, contact info, university, photo).
- Trip Details Input: Input flight time, number of bags, and destination airport.
- Rider/Driver Matching: System will suggest other users departing from the same campus at similar times to the same airport using a generative AI tool to match rider to driver.
- Match Display: Display matched users with key details: photo, name, flight time, number of bags.
- In-App Chat: In-app chat functionality for matched users to coordinate.
- Cost Estimator: Display estimated cost split per person.
- Flagging System: Allow users to flag others with whom the ride was unsafe or unpleasant. Flagged users are penalized in the matching algorithm.

## Style Guidelines:

- Primary color: Medium aquamarine (#7FFFD4), suggesting travel and coordination. 
- Background color: Pale aquamarine (#E2FFFA).
- Accent color: Pale cyan (#AFEEEE), for interactive elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headers, and 'Inter' (sans-serif) for body text.
- Simple, geometric icons for flight details, bag count, and chat.
- Clean and spacious layout with clear visual hierarchy.
- Smooth transitions and subtle animations when loading matches and messages.
